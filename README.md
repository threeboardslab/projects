# threeboardslab — projects

Open-source code for builds published at **[threeboardslab.com](https://threeboardslab.com)**.

Each folder is one project. Pick a folder, read its `README.md` for parts /
wiring / how it works, and copy the code into Arduino IDE, PlatformIO, or
your editor of choice.
